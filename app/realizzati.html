🛰️ Manual de Integrações e Rastreamento

Documentação operacional das tags do contêiner e pixels do site Realizzati.

1. Google Tag Manager (Contêiner GTM-55PTFQ2W)

O Tag Manager está instalado na Landing Page e no site institucional. As Tags de conversão disparam automaticamente ao mapear eventos do dataLayer como lead_submit_success (quando o formulário de orçamento é completado) e whatsapp_click (ao clicar no botão flutuante de chat).

2. API de Conversões do Meta (Pixel & CAPI)

O monitoramento é híbrido. O Pixel do Facebook é carregado no navegador do usuário e se comunica diretamente com a API do servidor da Meta para mitigar a perda de dados ocasionada por bloqueadores de anúncios (AdBlock). Envia os eventos padrão de Lead e Contact associados aos dados do usuário (Enhanced Conversions).

3. URL Tracking & UTMs

Todas as origens de tráfego pago (Google Ads, Meta Ads) incluem parâmetros de campanha estruturados. O script de captura de leads armazena localmente os parâmetros da URL no formulário oculto do usuário, garantindo que cada lead gravado no CRM possua a identificação exata de qual anúncio ou palavra-chave originou a conversão.
