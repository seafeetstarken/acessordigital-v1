# 🎨 Acessor Digital — Exploração de Identidade Visual

> **Briefing:** Moderno + Tecnológico + Custo-Benefício + Povão
> A marca precisa ser **inteligente sem ser elitista**, **tech sem ser fria**, **acessível sem parecer barata**.

---

## O Desafio

A Acessor Digital precisa falar com **dois mundos ao mesmo tempo:**

| Lado Tech | Lado Popular |
|-----------|-------------|
| IA, automação, dashboards | Dono de lanchonete, clínica, loja |
| Dados, métricas, performance | "Quanto vou vender a mais?" |
| WhatsApp Business API | "Meu WhatsApp responde sozinho?" |
| Meta Ads, pixel, conversão | "Vou aparecer mais no Instagram?" |

O segredo é comunicar **tecnologia como resultado**, não como complexidade.

### Referências de marcas BR que acertaram esse tom

| Marca | Cor | O que comunicam |
|-------|-----|-----------------|
| **Nubank** | Roxo | "Banco digital pra todo mundo" — tech + democratização |
| **iFood** | Vermelho | "Pedir comida é fácil" — tech + energia + ação |
| **PicPay** | Verde neon | "Dinheiro digital simples" — growth + acessível |
| **99** | Amarelo/Laranja | "Transporte pra todos" — energia + acessibilidade |
| **Mercado Livre** | Amarelo | "Compra e venda pra todo o Brasil" — alcance + confiança |
| **Hotmart** | Laranja → gradiente | "Negócio digital pra qualquer pessoa" — empreendedorismo |

> [!IMPORTANT]
> Padrão claro: marcas que falam com o "povão" brasileiro usam **cores quentes e vibrantes** — não azul corporativo frio. A cor precisa transmitir **energia, resultado e acessibilidade**.
